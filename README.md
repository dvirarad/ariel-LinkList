# 🎮 פלטפורמת משחקים לילדים - למידת קריאה וכתיבה בעברית

פלטפורמה אינטראקטיבית ומהנה שמלמדת ילדים בגילאי 5-8 לקרוא ולכתוב בעברית דרך משחקים חינוכיים.

## ✨ תכונות

### 🎯 6 משחקים שונים:

1. **🔤 זיהוי אותיות** - לימוד וזיהוי אותיות האלף-בית העברי
2. **🧩 בונים מילים** - בניית מילים פשוטות מאותיות
3. **🎴 משחק זיכרון** - מציאת זוגות של מילים
4. **📖 קוראים הברות** - חלוקת מילים להברות
5. **✏️ כותבים אותיות** - תרגול כתיבה על לוח ציור אינטראקטיבי
6. **🔊 התאמת צלילים** - זיהוי אותיות לפי צלילים

### 🌟 מאפיינים טכניים:

- ✅ ממשק משתמש ידידותי לילדים
- ✅ עיצוב צבעוני ומושך עם אנימציות
- ✅ תמיכה מלאה ב-RTL (עברית)
- ✅ מערכת ניקוד וכוכבים למעקב אחר התקדמות
- ✅ משוב חיובי ומעודד לכל פעולה
- ✅ Responsive - מתאים למחשב וטאבלט
- ✅ ללא צורך בהתקנה - פועל ישירות בדפדפן

## 🚀 איך להשתמש?

1. פתח את הקובץ `index.html` בדפדפן
2. בחר משחק מהתפריט הראשי
3. שחק ותלמד!

## 💻 טכנולוגיות

- HTML5
- CSS3 (עם Flexbox ו-Grid)
- JavaScript (Vanilla - ללא ספריות חיצוניות)
- Canvas API (למשחק הכתיבה)

## 📁 מבנה הפרויקט

```
├── index.html          # עמוד ראשי
├── styles.css          # עיצוב
├── main.js            # לוגיקה ראשית
└── games/             # תיקיית משחקים
    ├── letters.js     # משחק זיהוי אותיות
    ├── wordBuilder.js # משחק בניית מילים
    ├── memory.js      # משחק זיכרון
    ├── syllables.js   # משחק הברות
    ├── writing.js     # משחק כתיבה
    └── sounds.js      # משחק צלילים
```
